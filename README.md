# Clock, Timer, and Watch App

This is a complete Android app with a beautifully polished clock, timer, and stopwatch built using Jetpack Compose and Material Design. 
It supports multiple modes including an elegant analog clock, a flip clock, a digital timer, and a liquid interface stopwatch. 

## Features
- **Analog Clock Mode**: A beautiful, glowing analog clock centerpiece.
- **Flip Clock Mode**: A highly customized digital flip style clock.
- **Timer**: Robust countdown timer with background notifications.
- **Stopwatch**: Precision stopwatch with lap tracking.
- **Always On Display**: Keeps screen awake while tracking time.
- **Notifications**: See the active time remaining right from your status bar.
